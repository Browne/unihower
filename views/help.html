Creating the topic

First navigate to the topic page and click on Create a topic

Once you have done that, you need to complete the form shown. Enter the name of your topic and then optionally enter a description.

Note: You can enter markdown to format your description

Troubleshooting

Make sure you have included a name for your topic. This field is mandatory.

Double checking

You should be taken to a new page that shows the name of your topic alongside an Eisenhower decision matrix. You should also have the option to edit or delete the topic

You can also check the main topic page to see if your topic is listed
